# FNP Data Analysis – Excel Dashboard

This project analyzes the sales data of **Ferns & Petals (FNP)** using Microsoft Excel.  
The goal of this analysis is to explore sales trends, customer behavior, and product performance through an interactive dashboard.

## Project Objective

The objective of this project is to transform raw sales data into meaningful business insights using Excel data analysis tools.

This helps businesses understand:

- Revenue trends
- Customer purchasing patterns
- Product performance
- Seasonal sales patterns

## Dataset Description

The dataset contains information about:

- Order ID
- Customer details
- Product category
- Occasion
- Order date
- Delivery date
- Revenue
- City

This data is used to analyze sales performance across different dimensions such as time, location, and product categories.

## Analysis Performed

The following analysis was performed using Microsoft Excel:

- Total revenue and total orders
- Monthly revenue trends
- Revenue by occasion
- Product category performance
- Customer spending behavior
- Order delivery time analysis
- City-wise sales distribution

## Tools & Techniques Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Data Visualization

## Key Insights

Some key insights discovered from the analysis:

- Sales increase during festival occasions like Anniversary and Holi
- Certain product categories generate the majority of revenue
- Customer purchasing behavior varies by city
- Seasonal trends significantly affect sales performance
